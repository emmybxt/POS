# PosAPI

Server developed using Nodejs, Express and Mongodb.

## Installation

```
$ npm install
```

## Database
[install mongodb](https://docs.mongodb.com/manual/installation/)


start the database 

``` 
$ mongod 
```

## Starting the server
```
$ npm start
```
